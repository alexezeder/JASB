# JASB

## Synopsis

JASB - is a Slack bot

## Requirements

 - Python3
 - pip
 - virtualenv
 - CLion

## Installation

```sh
$ cd path/to/JASB
$ sudo apt install python3 python3-pip
$ pip3 install virtualenv
$ virtualenv venv
$ source venv/bin/activate
$ pip3 install -r requirements.txt
$
$ # don't forget to execute
$ deactivate
$ # when you finished you work with JASB
```
